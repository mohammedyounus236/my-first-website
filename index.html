Developing a website

The first step to creating a website is to create a folder. Once, the folder has been created, create 3 folders inside of it. These 3 folders are:

  • css
  • js
  • images

After the folders have been created open Sublime Text. Once sublime has opened:

  1. drag your folder into sublime to open it as a project
  2. Create a new file - file -> new
  3. Save the file in the root of the website folder as index.html
